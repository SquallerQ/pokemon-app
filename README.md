## Project Evolution

This project was developed step-by-step, with each stage in a separate branch:

1. **`class-components`** ‚Äî Initial version built with React class components
2. **`unit-testing`** ‚Äî Added unit testing with Vitest
3. **`hooks-and-routing`** ‚Äî Migrated to React Hooks and added routing
4. **`app-state-management`** ‚Äî Implemented state management
5. **`api-queries`** ‚Äî Integrated API queries for fetching Pokemon data
6. **`main`** ‚Äî Final version with Next.js and Server-Side Rendering ‚ú® (merged from `nextjs-ssr` branch)

## Features

- Server-Side Rendering (SSR) with Next.js
- Unit testing with Vitest
- Dark/Light theme toggle
- Internationalization (i18n) with multi-language support
- Modern React Hooks architecture
- API integration for Pokemon data
- Responsive design

## Run Locally

1. Clone the repo: `git clone https://github.com/SquallerQ/pokemon-app.git`
2. Navigate to the project directory: `cd pokemon-app`
3. Install dependencies: `npm install`
4. Start development server: `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000) in your browser
